## 📊 Fashion Store Data Analysis

#### 📌 Project Overview

This project analyzes sales data from a fashion retail store to uncover key business insights such as customer behavior, sales trends, product performance, and revenue patterns. The analysis is performed using Excel, focusing on data cleaning, transformation, and visualization.

#### 🎯 Objectives

Identify top-performing products and categories
Analyze monthly and seasonal sales trends
Understand customer demographics and purchasing behavior
Evaluate sales channels and order distribution
Generate actionable insights for business decision-making


#### The dataset includes:

Order ID
Customer details (gender, age group)
Product category
Order date
Sales amount
Order status
Sales channel (online/offline)

#### 🛠 Tools & Technologies
Microsoft Excel
Pivot Tables
Charts & Dashboards

#### 🔄 Data Cleaning Steps

Removed duplicate entries
Handled missing/null values
Standardized column formats (dates, categories)
Corrected inconsistent category names
Verified data types for calculations

#### 📊 Key Insights

Certain categories (e.g., ethnic wear, western wear) dominate revenue
Female customers contribute a higher percentage of total sales
Most sales occur during specific months (seasonal spikes)
Online channels outperform offline sales
A small number of products drive a large portion of revenue (Pareto effect)
